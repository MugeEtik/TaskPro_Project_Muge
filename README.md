# End-to-End QA Testing for a Kanban-Based Task Management Web Application

Comprehensive end-to-end QA testing project for a Kanban-based task management web application. 
Includes Test Plan, Requirement Traceability Matrix, 52 core Test Cases, Test Suites, and full Test Execution across UI (Desktop/Mobile) and API (Postman) workflows. Covers 23 API endpoints, 26 UI acceptance tests, bug reporting (Jira), severity/priority classification, and a final Test Summary Report with key metrics and findings.
